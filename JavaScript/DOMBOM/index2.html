蓝桥云课

document.getElementById() 通过 id 获取元素。 document.getElementByTagName() 通过标签名获取元素。 document.getElementByClassName() 通过 class 获取元素。 document.getElementByName() 通过 name 获取元素。 document.querySelector() 通过选择器获取第一个元素。 document.querySelectorAll() 通过选择器获取所有元素。 document.createElement() 创建元素节点。 document.createTextNode() 创建文本节点。 document.write() 输出内容。 document.writeln() 输出内容并换行。
